## 🧪 Quick Start (Node.js)

```js
const axios = require('axios');

const response = await axios.get('https://api.scrapingdog.com/amazon/product', {
  params: {
    api_key: 'YOUR_API_KEY',
    domain:'com',
    asin: 'B00AP877FS'
  }
});

console.log(response.data);
```
